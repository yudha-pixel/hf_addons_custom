🚚 FOTS Deposit Manager

Security Deposit Tracking & Daily Reconciliation for Traffic Hawkers

v1.0.0 Odoo 19.0 LGPL-3

📋 Overview

The FOTS Deposit Manager module provides comprehensive security deposit tracking and daily reconciliation management for Traffic Hawkers (FOTS Team) at Heritage Foods Limited.

This module streamlines the entire workflow from deposit collection to daily sales reconciliation, ensuring accurate tracking of cash, stock, and deposits.

✨ Key Features

👥 Hawker Management

  • Complete profile management
  • Auto-generated unique codes
  • Real-time deposit tracking
  • Stock monitoring
  • Status management

💰 Deposit Tracking

  • Deposit transactions
  • Refund processing
  • Deduction management
  • Balance validation
  • Complete audit trail

📊 Daily Reconciliation

  • Stock issued/returned tracking
  • Cash collection verification
  • Variance detection
  • Approval workflow
  • Automatic calculations

🔐 Security & Access

  • FOTS Manager role

  • User-based filtering

🚀 Installation

  1. Navigate to Settings → Apps → Update Apps List
  2. Search for "FOTS Deposit Manager"
  3. Click Install
  4. Verify FOTS Deposits menu appears

📦 Dependencies

  • stock - Inventory Management
  • point_of_sale - Point of Sale
  • product - Product Management

💡 Quick Start

  1. Create Hawker: FOTS Deposits → Hawkers → Create
  2. Record Deposit: FOTS Deposits → Deposits → Create (₦40,000)
  3. Activate Hawker: Click "Activate" button
  4. Daily Reconciliation: Click "Daily Reconciliation" button
  5. Enter Data: Stock issued, returned, cash collected
  6. Process: Click "Process Reconciliation"

🎯 Use Cases

  • Security Deposits: Track ₦40,000 deposits per hawker
  • Daily Sales: Reconcile stock sold vs. cash collected
  • Variance Management: Flag and approve discrepancies
  • Refunds: Process deposit refunds when hawkers leave
  • Deductions: Deduct for damages or losses
  • Reports: View hawker performance and statistics

📈 Benefits

  • ✅ Accountability: Every hawker's deposits and sales tracked
